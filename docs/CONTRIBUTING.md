# How to Contribute to This Project

## Step 1: Clone the Repository (First Time Only)
```bash
git clone https://github.com/Meadou/PokeCatch.git
```

## Step 2: Before Starting Your Work
Always update your local copy first:
```bash
git pull origin main
```

## Step 3: Create Your Branch
Create a branch with your name and task:
```bash
git checkout -b yourname-taskname
# Example: git checkout -b marjhun-pokemon-class
```

## Step 4: Do Your Work
- Only edit the files assigned to you
- Test your code before committing
- Add comments to your code

## Step 5: Commit Your Changes
```bash
git add .
git commit -m "Brief description of what you did"
# Example: git commit -m "Added Pokemon class with basic attributes"
```

## Step 6: Push Your Branch
```bash
git push origin yourname-taskname
```

## Step 7: Create Pull Request
1. Go to GitHub repository page
2. Click "Pull requests" tab
3. Click "New pull request"
4. Select your branch
5. Click "Create pull request"
6. Add description of your changes
7. Wait for review and merge

## Important Rules:
- Never push directly to `main` branch
- Never merge pull requests yourself
- Always create a new branch for your work
- Always `git pull` before starting
- Only edit your assigned files
